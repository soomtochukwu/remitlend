# RemitLend – Mobile Web3 Landing Page Design

## Project Overview

RemitLend's landing page is the gateway to the platform — a mobile-first experience designed to convert first-time visitors into active participants. The design introduces users to the decentralized financial ecosystem while emphasizing its integration with the Stellar Network, balancing high-impact visuals with clear, benefit-driven messaging.

---

## Key Design Contributions

### Hero Experience & Conversion

Designed the mobile hero section to establish trust and drive action immediately on load:

- Primary CTA: "Enter the Citadel" — bold, full-width button above the fold
- Social proof metrics displayed prominently: $1.2B+ TVL and 4.8% Avg. Yield
- Minimal copy, maximum signal — users understand the value proposition within seconds

### The RemitLend Arsenal — Feature Suite

Developed a scannable vertical feature list highlighting core value propositions with custom iconography and concise benefit-driven copy:

- Lend to Earn — passive yield on deposited assets
- Gamified Quests — XP rewards tied to financial actions
- Secure Vaults — audited smart contract infrastructure

Each feature card uses a two-column layout (icon + copy) optimized for thumb-zone interaction on mobile.

### Trust & Infrastructure Messaging

Integrated a "Verified Growth" section to reinforce platform credibility:

- Certified Audit badges from third-party security firms
- Stellar Network callout — low fees, high-speed settlement
- On-chain transparency messaging to address Web3 user skepticism

### Mobile-Optimized Layout

Crafted a seamless vertical scroll experience built for smaller screens:

- Sticky bottom navigation bar with four tabs: Citadel, Vaults, Quests, Empire
- Thumb-friendly tap targets throughout
- Section-based scroll with clear visual breaks between content zones
- No horizontal scroll — all content fits within a single column

### Urgency-Driven Onboarding

Designed the "The Gates are Opening" closing module to drive user acquisition:

- Simplified "Claim Access" flow — single input, single CTA
- Exclusivity framing to create FOMO and community pull
- Community size indicator to reinforce social proof at the point of conversion

---

## Design Principles

- Mobile-first, always — every decision optimized for small screens before desktop
- Trust before action — credibility signals appear before any CTA
- Clarity over cleverness — copy is direct, jargon is minimized
- Consistent gamification language — landing page tone matches the in-app experience

## Core UI Sections

### Hero Section
- Full-bleed background with subtle particle or glow effect
- Platform name + tagline
- TVL and Avg. Yield metrics in large type
- "Enter the Citadel" primary CTA button

### Arsenal Feature List
- Section header: "Everything You Need to Grow"
- 3 feature cards: Lend to Earn, Gamified Quests, Secure Vaults
- Icon + headline + one-line description per card

### Verified Growth Section
- Audit badge row (logos + "Certified" label)
- Stellar Network integration callout with logo
- Short trust statement

### Sticky Bottom Nav
- 4 tabs: Citadel, Vaults, Quests, Empire
- Active state: teal underline + icon highlight
- Fixed to bottom of viewport on scroll

### Gates are Opening — Closing Module
- Urgency headline
- Community size counter
- Email / wallet input
- "Claim Access" CTA button
